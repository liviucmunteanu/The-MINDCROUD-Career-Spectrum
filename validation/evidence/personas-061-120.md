# MINDCROUD Career Spectrum — Validation Personas 61–120

Instrument facts assumed by all expected outcomes:
- 5-point Likert (1 = Strongly disagree … 5 = Strongly agree), numeric-only answer UI.
- 60 scored items (10 per avatar; MAK/MEN/EXP have 2 reverse-keyed items each, GRD/VIS/PIO have 3).
- Spectrum Score per avatar = (item mean − 1) / 4 × 100 (all 5s = 100, all 4s w/ reverses at 2 = 75, all 3s = 50).
- Attention checks: Q17 correct = 4 (Agree), Q33 correct = 1 (Strongly disagree), Q49 correct = 3 (Neutral).
- Consistency pairs: Q1↔Q41 (teaching, MEN), Q10↔Q56 (detail-checking, GRD); flag on large absolute discrepancy.
- Dominant Avatars = top 2 scores, plus 3rd if within 10 points of score #2.
- Differentiation Index (DI); flat-profile flag when DI < 15.
- Verdict vocabulary used below: VALID / VALID-WITH-FLAG (interpret with caution) / REVIEW (questionable) / INVALID.
- RIASEC hexagon: adjacent pairs MAK–EXP, EXP–VIS, VIS–MEN, MEN–PIO, PIO–GRD, GRD–MAK; opposite pairs MAK–MEN, EXP–PIO, VIS–GRD.

Template per persona:
ID/Name — demographics — career context — Big Five — drivers — PURPOSE — TENDENCY (incl. attention-check & consistency-pair behavior) — EXPECTED OUTCOME (Dominant Avatars / DI / flags / verdict).
